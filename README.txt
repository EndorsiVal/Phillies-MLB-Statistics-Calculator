
Run the main.exe executable to access the program.

Alternatively it can be run from source by following these steps below:

- You must have Python 3.9 or greater installed. It can be gotten from here https://www.python.org/downloads/ .

- Use 'pip install -r requirements.txt' in the command shell to get all dependencies that might not be installed already.

- Then to run the project use in the command shell 'python main.py'.
